Homebrew

homebrew 安装

因为game porting toolkit需要，所以先安装homebrew的x86_64版本。

which brew 查看是否安装

切换arm / x86_64版本

想进入x86模式，请执行：arch -x86_64 zsh

homebrew 换国内源

常用指令

brew update 更新brew

brew upgrade 更新所有软件

brew search 搜索软件

brew install 安装软件

brew uninstall 卸载软件

brew list 列出已安装软件

brew info 查看软件信息

brew deps 查看软件依赖

brew outdated 查看过时软件

brew pin 锁定软件，防止更新

brew unpin 解锁软件，允许更新
